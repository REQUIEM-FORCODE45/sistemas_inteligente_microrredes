# Sistema Inteligente de Microrredes

Este repositorio contiene el **backend** y el **frontend** de un sistema inteligente para la gestión y control de microrredes eléctricas. El sistema utiliza técnicas de inteligencia artificial (IA) para optimizar el funcionamiento, el monitoreo y la toma de decisiones en microrredes.

## Descripción General

El proyecto está dividido en dos partes principales:

- **Backend:** Proporciona la lógica de negocio, algoritmos de IA, control predictivo (MPC) y la gestión de datos. Expone una API para la comunicación con el frontend y otros sistemas.
- **Frontend:** Interfaz gráfica de usuario (GUI) desarrollada para visualizar el estado de la microrred, interactuar con el sistema y mostrar resultados de las simulaciones y optimizaciones.

## Características Principales

- Optimización del flujo de energía en la microrred.
- Implementación de control predictivo basado en modelos (MPC).
- Integración de algoritmos de inteligencia artificial para la toma de decisiones.
- Visualización en tiempo real de datos y resultados.
- API REST para integración y comunicación entre componentes.
